# Port Tracking System - Backend

## Description
Backend application for container tracking and port logistics management.

## Technologies
- Java 17
- Spring Boot 3.3.5
- Spring Security
- JWT Authentication
- Spring Data JPA
- Hibernate
- MySQL

## Features
- Authentication & Authorization
- Role Management (SUPER_ADMIN, ADMIN, CLIENT)
- Container Management
- Container Tracking
- Geolocation
- Audit Logs
- Notifications

## Architecture
Layered Architecture:
- Controller
- Service
- Repository
- Database
